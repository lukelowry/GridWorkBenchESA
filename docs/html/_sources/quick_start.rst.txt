Quick Start
===========
The following quick start example uses the IEEE 14 bus case, which can
be from `Texas
A&M University
<https://electricgrids.engr.tamu.edu/electric-grid-test-cases/ieee-14-bus-system/>`__.
Notes related to the case are available from the `University of Washington
<https://labs.ece.uw.edu/pstca/>`__.

You can find API documentation
`here <https://wyattlaundry.github.io/GridWorkBenchESA/html/>`__ and more examples
`here <https://wyattlaundry.github.io/GridWorkBenchESA/html/snippets.html>`__.

.. include:: snippets/quick_start.rst

For more examples, you can find them `here <https://wyattlaundry.github.io/GridWorkBenchESA/html/snippets.html>`__.
